03 / 06
Model layer

Ensemble deep learning

LSTM

Stacked bidirectional LSTM with attention — multi-step forecasts from weekly feature mart.

TFT

Temporal Fusion Transformer via pytorch-forecasting — quantile loss for P10/P50/P90.

N-BEATS

Interpretable basis expansion — strong baseline for univariate and global models.

Ensemble

Weighted combination of model outputs with MLflow-tracked experiments and benchmark evaluation in src/evaluation/.

04 / 06
Modern stack

Production MLOps

Original

SQL Server · manual SQL · R forecast package · batch CSV · no serving or monitoring.

Upgraded

dbt + DuckDB · Airflow DAGs · MLflow registry · FastAPI + Redis · Prometheus drift monitor · Kafka ingestion.

Sources → Kafka / CDC → dbt (staging → marts) → feature mart → PyTorch Lightning training (LSTM, TFT, N-BEATS) → MLflow registry → FastAPI forecast-api (:8000) + Redis cache
